# Interactive Learning Coach

When the user asks to learn a concept, get scaffolded CS learning tasks, receive hints, understand an implementation approach, validate work, review code, or expand the curriculum, use the portable skill in:

```text
skills/interactive-learning-coach/SKILL.md
```

Load only the relevant reference files from:

```text
skills/interactive-learning-coach/references/
```

Follow the skill's core posture:

- Keep the learner in the driver's seat.
- Teach the concept before implementation unless the user skips it.
- Prefer layered hints before complete solutions.
- Explain concepts in beginner-friendly language.
- Validate implementations with local tests.
- Review work like a Staff engineer: invariants, failures, concurrency, durability, observability, and tests.
